Meine Git Anleitung

Git Befehle:

Befehl Bedeutung
git init Erstelt ein Repository lokal
git add macht bereit für schnapschuss
git commit -m"" macht schnappschuss und gibt in "" namen von veränderungen an
git checkout bewegt sich zu angegebenen Bereich
git merge bringt verschiedene branches zusammen
git branch Zeigt die branches an im repository wo man sich befindet, *welches man gerade benutzt
git log Zeigt was im Repository passiert ist
git push repository wird von lokal augf remote verschickt, eg. auf Github
git pull repository wird von remote auf lokal verschickt
git reset --hard[commit Nr.] Erstelt ein Repository lokal
git push --force um resettet commit auf remote eg. Github hochzuladen
git remote kannst du Verbindungen mit anderen Repositorys erstellen, abrufen und löschen.
